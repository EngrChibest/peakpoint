# Peak Point International School Portal

A premium, institutional-grade web portal for **Peak Point International School (PPIS)**. Built with modern web technologies to provide a seamless experience for parents, students, and staff.

## 🌟 Key Features

- **Admissions Hub**: A complete guide for prospective parents, including a 7-step enrollment process, age requirements table, and downloadable resources.
- **Campus Life & Facilities**: Detailed overview of school facilities, laboratories, and extracurricular clubs.
- **Institutional Governance**: Clear presentation of school policies, safeguarding statements, and codes of conduct.
- **Dynamic News & Updates**: Categorized news system to keep the community informed about academic achievements and upcoming events.
- **Interactive FAQ**: A centralized, user-friendly accordion system for common inquiries on Admissions and Contact pages.
- **Modern Contact System**: integrated contact forms and institutional information.

## 🛠️ Technology Stack

- **Frontend**: React 19 + Vite
- **Styling**: Tailwind CSS (Vanilla)
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Routing**: React Router 7

## 🚀 Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/peakpoint-portal.git
   ```
2. Navigate to the project directory:
   ```bash
   cd peakpoint
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

The app will be available at `http://localhost:5173`.
